# Core Instructions

1. Security; always implement appropriate steps to ensure the website follows required industry-standard security measures.
2. Performance; website must always be performant and snappy.
3. Creative; website must be creative and not copy common "vibe coded" websites.
4. Optimized; backend should be optimized to reduce hosting costs and issues with providers.
5. Curl-friendly; website must always respect that it could be navigated through the curl feature.
