# lsalik.dev tracker

## BUGS
**background animation doesn't span the entire screen**
- Issue description: the animation in the background doesn't stretch out enough to the right to completely fit the screen
- Intended behavior: the animation fully encompases the entire stretch of the screen

**clicking through the different nav pages after a theme switch, reverts quickly to the default theme then back**
- Issue description: if switching from the default theme to another, switching pages causes the default theme to very shortly to appear then switch back to the selected theme
- Intended behavior: smooth and seamless switching between pages with the selected theme remaining

**projects cards have no border**
- Issue description: projects cards in the projects page have no border
- Intended behavior: projects cards should have borders like blogs cards do

**~ in breadcrumb  is highlightable**
- Issue description: hovering the mouse over ~ in the breadcrumb highlights it like if it was clickable
- Intended behavior: ~ should not be highlightable

**prompt path in breadcrumb is highlightable**
- Issue description: however the mouse over the prompt path (e.g., "blogs" when the blog page is selected) makes it highlightable and clickable
- Intended behavior: there's no point in clicking in the prompt path since you already are on that page, so no need to make it highlighted and clickable

**double hashtags (##) headings appear on the browser with their hashtag **
- Issue description: the HTML code successfully shows those headings as h2, but the hashtags are visible in the browser.
- Intended behavior: like h1 tags, who only use one #, the hashtags should not be visible in the browser.

**page-main not stretched out to the sides enough**
- Issue description: the page-main platform with all the website content doesn't stretch enough to the sides. This leaves a large amount of whitespace on the website.
- Intended behavior: the page-main platform should stretch out slightly more to the sides. It should also have a slight amount of opacity, about 0.9 for now

**curl return is too packed on the top**
- Issue description: when using curl, the return is too close to the command prompt.
- Intended behavior: add a new line to the top of the curl return, so it stays spaced out from the command prompt.
