# inspirations

The following references were used as inspiration and code-review for the website:

- [ysap.sh](https://ysap.sh) — terminal-first personal site with full curl support. The curl demo on the browser version, ANSI blocks logo and curl support used `ysap.sh` as reference.
- [pretext](https://github.com/chenglou/pretext) — specifically for the ASCII background animation. The Variable Typographic ASCII demo was used as reference.
